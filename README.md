# Calificador y seguimiento de Moodle

Aplicación web estática para analizar exportaciones del calificador de Moodle y cruzarlas con la página de Participantes del curso.

## Funciones principales

- Procesamiento local: los datos no se envían a un servidor.
- Filtros visibles por participante, riesgo, promedio, señal y estado de Moodle.
- Conversión de “Último acceso” a días decimales.
- Ordenamiento ascendente y descendente en todas las columnas del seguimiento.
- Exportación del informe a Excel.
- Diseño embebible en Google Sites.

## Publicación

El workflow de GitHub Pages publica automáticamente `index.html` cuando se actualiza la rama `main`.
